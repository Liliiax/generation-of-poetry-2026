# generation of poetry

## Описание

**Генерация поэзии** $-$ исследовательский проект, посвященный сравнению современных нейросетевых архитектур (**LSTM, Transformer и Mamba**) для генерации русскоязычных стихотворений.

Цель исследования — определить, какая из архитектур лучше справляется с генерацией поэзии с учетом ритма, рифмы и ударений на основе автоматических метрик и субъективной экспертной оценки.

Результатом проекта является веб-приложение, позволяющее генерировать стихотворения с использованием одной из обученных моделей.

## Возможности

- **Генерация стихотворений** $-$ создание стихотворений с использованием одной из трех обученных моделей (LSTM, Transformer или Mamba).

- **Выбор модели** $-$ возможность выбрать архитектуру нейронной сети для генерации.

- **Настройка генерации** $-$ выбор количества генерируемых стихотворений и их максимальной длины.

- **Визуализация результатов** $-$ отображение сгенерированных стихотворений в веб-интерфейсе.

## Используемый датасет

В проекте используется датасет **Ars Poetica**, содержащий русскоязычные стихотворные тексты с разметкой ударений.

Для обучения моделей используется поле **accentuation**, в котором тексты уже содержат расставленные ударения, что позволяет учитывать особенности ритмической организации стихотворений.

Корпус включает **8498** стихотворений.

Преимущества датасета:

- наличие разметки ударений
- подходит для char-level моделей
- позволяет сохранять поэтический стиль и ритмическую структуру
- не требует дополнительной предобработки ударений

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/Liliiax/generation-of-poetry-2026.git
cd generation-of-poetry-2026
```

### 2. Создание виртуального окружения

#### Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### 3. Запуск приложения

```bash
streamlit run app.py
```
